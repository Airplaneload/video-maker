# MicroFlight - 视频帧导出与批量抠图工具

MicroFlight 是一款视频帧导出与批量抠图桌面工具，支持目标框跟踪、联系图批量抠图、下落速度计算等功能。

## 使用方式

双击 `MicroFlight.exe` 启动程序。exe 与 `_internal` 文件夹必须在同一目录下。

### 导出所有帧

1. 选择视频文件。
2. 选择帧图片的导出文件夹。
3. 选择 JPG 或 PNG；JPG 可设置质量，默认 95。
4. 点击"导出视频所有帧"。

输出文件按 `frame_000000.jpg` 或 `frame_000000.png` 连续编号。

### 框选批量抠图

1. 打开"框选批量抠图"页并载入视频。
2. 在首帧点击"自动框选当前帧"，或选择"拖框"后手动框选。
3. 点击"跟踪下一帧"逐帧检查，或"自动跟踪到末帧"批量生成目标框。
4. "框轨迹总览"会把不同帧的目标框覆盖到第一帧上，便于检查运动路径。
5. 设置每页帧数，点击"生成本页联系图并统一抠图"。多帧裁剪块会排成一张大图统一分割。
6. 在联系图或单帧上使用前景笔、背景笔涂画，再点击"当前框抠图/笔刷精修"。
7. 点击"应用联系图到各帧"，逐帧检查后导出透明 PNG。
8. "选择帧并生成抠图轨迹总览"可逐帧选择要显示的抠图，也可以设置起始帧和间隔自动抽帧。

编辑画面时按住 Ctrl 滚动鼠标滚轮可围绕鼠标位置缩放；放大后用鼠标中键或右键拖动画面，点击"适应窗口"恢复全图。

### JSON 项目

1. 选择原始视频后点击"新建 JSON 项目"。
2. 指定项目 `.json` 的保存位置。
3. 程序会创建同名 `_assets` 资源目录，并把视频复制到项目中。
4. 以后点击"导入 JSON 项目"，明确选择需要继续编辑的项目。

项目会保存目标框、逐帧掩膜、联系图、当前帧、标尺、速度结果、轨迹选帧和显示参数。

### 下落速度

1. 选择"标尺"工具，在画面中拖出一段已知真实长度。
2. 输入标尺实际长度并选择 μm/mm/cm/m 单位。
3. 完成目标框跟踪或抠图后点击"计算下落速度"。

程序优先使用抠图区域质心，没有掩膜时使用目标框中心，并结合视频 FPS 计算平均下落速度和区间最大速度。

## 系统要求

- Windows 10 或更高版本
- 无需安装 Python 或其他依赖，开箱即用
