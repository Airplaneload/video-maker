# 视频帧导出与批量抠图工具

程序支持导出视频全部帧，以及通过目标框跟踪和联系图完成批量抠图。

## 图形界面

双击 `run_gui.pyw`，或运行：

```powershell
C:\Users\wei\anaconda3\python.exe -m microflight.gui
```

### 导出所有帧

1. 选择视频文件。
2. 选择帧图片的导出文件夹。
3. 选择 JPG 或 PNG；JPG 可设置质量，默认 95。
4. 点击“导出视频所有帧”。

程序会实时显示当前帧、帧号和总体进度。输出文件按
`frame_000000.jpg` 或 `frame_000000.png` 连续编号。

### 框选批量抠图

1. 打开“框选批量抠图”页并载入视频。
2. 在首帧点击“自动框选当前帧”，或选择“拖框”后手动框选。
3. 点击“跟踪下一帧”逐帧检查，或“自动跟踪到末帧”批量生成目标框。
4. “框轨迹总览”会把不同帧的目标框覆盖到第一帧上，便于检查运动路径。
5. 设置每页帧数，点击“生成本页联系图并统一抠图”。多帧裁剪块会排成一张大图统一分割。
6. 在联系图或单帧上使用前景笔、背景笔涂画，再点击“当前框抠图/笔刷精修”。
7. 点击“应用联系图到各帧”，逐帧检查后导出透明 PNG。
8. “选择帧并生成抠图轨迹总览”可逐帧选择要显示的抠图，也可以设置起始帧和间隔自动抽帧。选中的透明目标会按原坐标覆盖到第一帧。

编辑画面时按住 `Ctrl` 滚动鼠标滚轮可围绕鼠标位置缩放；放大后用鼠标中键或右键拖动画面，点击“适应窗口”恢复全图。画笔拖动时使用轻量预览，松开后才进行完整遮罩渲染。

“掩膜填充透明度”可调节绿色覆盖强度，默认 18%。精细编辑时可点击“仅看轮廓”，隐藏绿色填充并保留黄色抠图边缘；“弱化背景”仅用于需要突出前景时，默认关闭。这些选项只影响预览，不影响实际掩膜和透明 PNG。

### JSON 项目

程序不再根据视频名称自动恢复项目：

1. 选择原始视频后点击“新建 JSON 项目”。
2. 指定项目 `.json` 的保存位置。
3. 程序会创建同名 `_assets` 资源目录，并把视频复制到项目中。
4. 以后点击“导入 JSON 项目”，明确选择需要继续编辑的项目。

项目会保存目标框、逐帧掩膜、联系图、当前帧、标尺、速度结果、轨迹选帧和显示参数。编辑时仍会自动增量保存，但只保存到当前明确导入的 JSON 项目。

旧版项目仍兼容。点击“导入旧版项目”，选择旧隐藏目录中的
`project.json`，再指定一个新的 JSON 项目位置。程序会复制原视频并迁移旧框、掩膜和设置，不会静默打开旧项目。

轨迹总览可通过“显示帧序号”决定是否标注帧号。背景亮度、背景饱和度、目标融合强度和边缘羽化均可手动设置，默认背景亮度为 100%，不会自动调暗。调整后点击“按当前参数重新生成”，确认满意后点击“保存轨迹总览图”输出 PNG 或 JPG。

### 下落速度

1. 选择“标尺”工具，在画面中拖出一段已知真实长度。
2. 输入标尺实际长度并选择 `μm/mm/cm/m` 单位。
3. 完成目标框跟踪或抠图后点击“计算下落速度”。

程序优先使用抠图区域质心，没有掩膜时使用目标框中心，并结合视频 FPS 计算平均下落速度和区间最大速度。画面向下为正方向。标尺和速度结果保存在 JSON 项目中。

鼠标停留在项目按钮、跟踪、抠图、标尺、速度和画布等主要控件上，会显示悬浮功能提示。

抠图页底部始终显示任务进度。视频载入、帧读取、框选、跟踪、联系图分割、轨迹合成和 PNG 导出都会显示当前阶段及百分比；批量任务按实际处理帧数更新。

联系图按页处理，避免长视频一次拼接造成内存不足。每个裁剪块都保留原帧编号和坐标，掩膜会自动映射回原视频帧。
